## 2. 분석 모델 및 로직 고도화 (Algorithm)
- **산출식 정밀 이식**: 기존 Gemini 챗봇과의 대화 이력을 분석하여 도출된 세부 산출식 데이터를 프로젝트(`WeightEngine` 등)에 체계적으로 이식.
- **분석 요약(yt_reason) 누락 진단**: 리포트에서 `yt_reason`이 충분히 반영되지 않거나 "파트 1/1" 등으로 끝나는 현상을 분석하여 긴 텍스트 전달 로직 최적화.
- **페르소나 기반 서머리 개선**: API 호출 시 전문 애널리스트 페르소나를 부여하여 분석 품질 고도화.

  - 애널리스트 분석 논조의 논리적 근거 및 권장 대응 항목 포함
