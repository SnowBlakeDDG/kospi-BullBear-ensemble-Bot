# 🧠 G-Ensemble v1.8 Core Algorithm Integrated Specification

이 문서는 G-Ensemble v1.8 모델의 가중치 산출, 수학적 공식, 시장 판정 기준 및 예외 처리 로직을 집대성한 통합 사양서입니다.

---

## 1. 핵심 지표 및 기본 가중치 (Base Weights)
모델은 4개의 메인 지표를 표준화하여 합산하며, 시장 상황에 따라 가중치를 동적으로 재배분합니다.

*   **F (외국인 수급)**: 40% (0.4) - 시장 방향성의 핵심 동력
*   **K (KORU ETF)**: 30% (0.3) - KOSPI 추종 레버리지 및 변동성
*   **H (인간 지표)**: 20% (0.2) - 커뮤니티/유튜브 센티멘트 기반 역발상(Contrarian)
*   **R (개인 수급)**: 10% (0.1) - 하방 지지력 및 과열 보조 지표

---

## 2. 지표 표준화 및 판정 기준 (Normalization & Labels)
모든 지표는 -1.0 ~ +1.0 사이의 연속적인 값으로 선형 보간(Linear Interpolation)됩니다.

### 2.1 수급 및 변동성 지표 (F, R, K)
$$Score = \text{clamp}\left(\frac{Value}{Threshold_{max}}, -1.0, 1.0\right)$$

| 판정 라벨 | 스코어 범위 | 외국인 (F) 범위 | KORU (K) 범위 | 비고 |
| :--- | :--- | :--- | :--- | :--- |
| **Extreme** | **+/- 1.0** | +/- 5,000억 초과 | +/- 4.5% 초과 | Strong Bull/Bear |
| **Standard** | **+/- 0.7** | +/- 3,000억~5,000억 | +/- 2.5%~4.5% | Bull/Bear |
| **Slight** | **+/- 0.3** | +/- 1,000억~3,000억 | +/- 1.0%~2.5% | Slightly Bull/Bear |
| **Neutral** | **0.0** | +/- 1,000억 미만 | +/- 1.0% 미만 | 박스권 횡보 |

### 2.2 인간 지표 (H) - 역발상 로직
$$H = -\left( \frac{Gemini\_Score - 50}{50} \right)$$
*   **Dead-zone**: 40~60점 사이는 $H = 0$ 처리 (미세 변동 노이즈 제거).
*   **역발상**: 극심한 낙관(100점) -> $H = -1.0$ / 극도의 공포(0점) -> $H = +1.0$.

---

## 3. 최종 스코어(S) 및 7단계 시장 판정
$$S = (W_F \times F) + (W_K \times K) + (W_H \times H) + (W_R \times R) + \alpha_{FX}$$

| 스코어 범위 (S) | 판정 라벨 (Label) | 시장 대응 전략 |
| :--- | :--- | :--- |
| **S ≥ +0.50** | **Strong Bullish** | 압도적 상방, 공격적 매수 |
| **+0.35 ≤ S < +0.50** | **Bullish** | 견고한 상승세 유지 |
| **+0.15 ≤ S < +0.35** | **Slightly Bullish** | 완만한 우상향, 순환매 장세 |
| **-0.15 < S < +0.15** | **Neutral** | 방향성 부재, 단기 박스권 |
| **-0.35 < S ≤ -0.15** | **Slightly Bearish** | 상단 제한, 리스크 관리 시작 |
| **-0.50 < S ≤ -0.35** | **Bearish** | 추세적 하락 진입, 비중 축소 |
| **S ≤ -0.50** | **Strong Bearish** | 패닉 셀링, 현금 비중 최대화 |

---

## 4. 동적 제어 및 보정 체인 (Dynamic Chains)

### 4.1 환율 보정 ($\alpha_{FX}$)
환율은 시장의 하방 압력으로만 작용 (0 또는 음수).
*   **0.0 (안정)**: $-0.3\% \sim +0.1\%$ 변동
*   **-0.1 (상승 주의)**: $+0.1\% \sim +0.5\%$ 상승 또는 1350원 돌파 시도
*   **-0.25 (급등 경고)**: $+0.5\%$ 초과 급등 또는 NDF 5원 이상 폭등
*   **-0.5 (로컬 리스크)**: 달러 인덱스 하락 중 원화만 약세 시 페널티 2배

### 4.2 볼러틸리티 오버드라이브
$V_{KORU} > (\sigma \times 1.5)$ 일 때 가동.
*   가중치 조정: $F(0.4 \to 0.5)$, $K(0.3 \to 0.2)$, $H(0.2 \to 0.1)$.
*   $K$ 판정 임계치를 $\pm 4.5\%$로 상향하여 노이즈 제거.

### 4.3 크라이시스 모드 (Crisis Mode)
VIX +15% 급등 또는 KORU 장초반 변동성 > 5% 시 진입.
*   **외국인 선물 60%**, 외국인 현물 20%, KORU 15%, 인간 지표 5%, **개인 0%**.
*   $\alpha_{FX}$ 감점 상한 해제 (무제한 감점).

## 5. 예외 및 신뢰도 관리

### 5.1 데이터 정합성 ($I_i$) 및 가중치 재배분
$$I_i = \max\left(0, 1 - \frac{\text{Latency}}{60}\right)$$
*   **지연 페널티**: 60초 이상 지연 시 가중치 0 처리.
*   **재배분 공식**: 지연이나 감쇄로 발생한 유휴 가중치($W_{vacant}$)는 정합성 지수에 비례하여 타 지표로 전이됨.
    $$W_{new}(i) = W_{old}(i) + (W_{vacant} \times \frac{I_i}{\sum I_j})$$

### 5.2 외국인 수급 질적 분해 (Cash vs Futures)
현물/선물 방향성 조합에 따른 $F$ 점수 보정 계수 ($W_{adj}$):
*   **Both Buy**: x 1.2 (상승 확정)
*   **Cash Buy / Fut Sell**: x 0.7 (헷지성)
*   **Cash Sell / Fut Buy**: x 0.5 (저가 매수 시도)
*   **Both Sell**: x 1.5 (하락 추세 가속)
*   **프로그램 매매**: 외국인 매수의 70% 초과 시 $F$ 신뢰도 **10% 감점**.

### 5.3 KOSPI-KORU 디커플링 ($D_{bias}$)
*   $| \Delta KORU - (3 \times \Delta KOSPI200) | > 1.2\%$ 시 $K$ 점수에 **x 0.8** 적용 (투기 희석).

### 5.4 인간 지표 3일 감쇄 (Time-Decay)
*   1일차(1.0) -> 2일차(0.7) -> 3일차(0.5) -> 4일차(0.0)로 영향력 축소.
*   감쇄된 비중은 위 5.1의 재배분 공식에 따라 실시간 데이터가 가장 정확한 지표로 자동 이전됨.

## 6. 추가 실행 디테일 (Execution Details)

### 6.1 야간 나이트-워치 및 Gap-Fill
*   **Gap-Fill**: 예상 갭상승률 > 1% 시, 장초반 15분간 전체 스코어에 **-0.2 페널티** 부여 (익절 매물 압력 반영).

### 6.2 오버슈팅 방지
*   $S \in [+0.15, +0.35]$ (Slightly Bullish) 이고 $H < 0$(낙관)이면 $S = 0.0$으로 강제 조정.

### 6.3 마이너리티 리포트 (Minority Report)
*   $S \ge +0.50$ 이나 단일 지표가 $-1.0$ (Extreme Bearish)인 경우 `[Warning: Divergence Detected]` 알럿 출력 및 반대 지표 원인 분석 우선 보고.
